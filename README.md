
# Cab Booking App

Created the complete backend of an online cab booking app. Design and functionalities are mostly inspired from Uber. 

## Technologies and tools used

 - The Go Programming Language
 - PostgreSQL
 - Gorilla web toolkit
 - Redis
 - JWT
 - GORM
 - Distance Matrix API
 - Razorpay Payment Gateway
